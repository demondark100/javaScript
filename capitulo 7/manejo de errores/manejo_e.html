manejo de errores

pseudocodogp:

ejecutarError{
pasa un error
}
mostrarError(error){
mostrar el error
}

javaScript:

try {
console.log(respuesta)
} catch (error) {
console.log("esta variable no esta declarada")
}

con esto podemos manejar los errores que puedan ocurrir en nuestro programa.

en este caso la variable "respuesta" no esta declarada y podemos manejarla mandando un mensaje.

con esto no controlamos los errores de sintaxis solo los errores que en teroaia podemos esperar.

finally

try {
console.log("hola mundo")
} catch (error) {
console.log("esto es un error")
} finally {
console.log("siempre me ejecutare haya o no errores.")
}

"finally" siempre se ejecutara si hay errores o no.

throw

try {
throw "tiraremos un error"
} catch (error) {
console.log(error)
}

"throw" sera para tirar errores directamente.
