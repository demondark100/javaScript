strokeRect / fillRect

ambos tienen los mismos parametros al mismo orden:
parametro 1 left
parametro 2 top
parametro 3 width
parametro 4 height

"strokeRect" es para hacer trazos / lineas.
"fillRect" es para rellenar el canvas.

strokeStyle / fillStyle

siempre deben ir antes de "strokeRect" y "fillRect".

"strokeStyle" es para darle color al trazo / linea.
"fillStyle" es para darle color al relleno.

lineWidth

esto es para darle grosor a la linea / trazo siempre debe ir antes de "strokeRect".
