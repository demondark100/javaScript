LocalStorage y SessionStorage

esta api se encarga de guardar la informacion de la pagina web.

LocalStorage:

esto guardara la informacion pero a la hora de cerrar la pagina web si se guardara la informacion a la hora de volver a entrar al sitio.

SessionStorage:

esto guardara la informacion de forma local osea que a la hora de cerrar la pagina la informacion se perdera.
