Drag y Drop

Drag y Drop es el arrastre y desplazamiento de los elementos , todos los objetos tienen drag osea que si arrastras cualquier objeto el ebjeto sentira el drag a la hora de desplazarlo.

dragstart

esto es el inicio del arrastre.

drag

esto es el desplazamiento.

dragend

esto es cuando el desplazamiento finaliza

desplaza el circulo y veras como aparece 1,2,3 posdata manten el click en el circulo por un momento para que funcione.

dragenter

esto verifica que el elemento este entrando

dragover

este verifica que se mueva dentro del elemento osea que el elemento que arrastramos siga dentro del otro elemento

drop

verifica cuando se suelta el elemento que estamos arrastrando

dragleave

esto verifica cuando el elemento se va osea se sale del elemento

abre la consola con F12 , arrastra el circulo hacia el rectangulo , mueve el circulo de adentro a afuera del rectangulo y suelta el circulo dentro del rectangulo.
