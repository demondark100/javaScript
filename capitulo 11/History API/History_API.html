History API

con el objeto "History" trabajaremos con el historial una api muy util.

console.log(history.__proto__);

con esto veremos el protoripo de este objeto.

back

history.back()

esto nos devolvera a una pagina atras.

forward

history.forward()

con esto iremos un sitio mas adelante de donde estamos.

length

history.length()

con esto veremos el tama√±o del historial.

go

history.go()

con esto recargaremos la pagina.

history.go(-1)

con esto recargaremos la pagina anterior a la que estamos.

history.go(1)

con esto recargaremos la pagina despues en la que estamos.

pushState

estas son entradas de historial por ejemplo si abrimos un sitio y modificamos la entrada como en el codigo y abres el historias se abrira esa entrada de historial.

state

console.log(history.state);

con esto veremos el nombre que pusimos en este caso "historial"

popstate

esto es como el state solo que nos evitaremos poner cada rato "history.state" a la hora de movernos en el historial.

replaceState

esto es lo mismo que "pushState" solo que no deja entradas en el historial.
