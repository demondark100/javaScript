bucles

con los bucles podremos recorrer arreglos o tambien hacer algo de una forma repetida.

con "++" ira incrementado un numero de 1 en 1.

bucle while:
pseudocodigo 1:
crear variable = valor;
bucle( condicional ) {
variable++;
recorrer arreglo o hacer algo.
}
bucle for:
pseudocodigo 2:
bucle ( crear variable = valor ; condicional ; variable++ ) {
recorrer arreglo o hacer algo.
}
bicle in:
pseudocodigo 3:
resumen 1:
bucle ( recorrer bucle y mostrar posisiones ) {
hacer algo.
}
bucle of:
pseudocodigo:
bucle ( recorrer bucle y mostrar el contenido ) {
hacer algo.
}

ahora con javaScript.

javaScript while:
let i = 0;
while(i < 5){
i++;
console.log("se ejecuto este bucle");
}
javaScript for:
for(let i = 0; i < 5; i++){
console.log("se ejecuto el bloque for");
}
javaScript bucle in:
let arreglo = [
"user1",
"user2",
"user3",
"user4"
];
for(let arr in arreglo){
console.log(arr);
}
javaScript bucle of:
let arreglo = [
"user1",
"user2",
"user3",
"user4"
];
for(let arr of arreglo){
console.log(arr);
