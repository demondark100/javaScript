para poder llamar a un objeto debemos hacer una variable de respaldo

pseudocodigo llamar:
variable = nuevo objeto(poner la descripcion del objeto)

ahora con sintaxis javaScript:

javaScript llamar:
const puerta = new Puerta("negro","madera")

ahora podemos llamar al objeto puerta de esta manera

console.log(puerta)

y si queremos algun dato en especifico podemos hacer esto:

console.log(puerta.color);
resultado: "negro"
console.log(puerta.materia);
resultado: "madera"

luego tenemos a los metodos que son igual que las funciones solo que al estar dentro de un objeto se le denomina como metodo.

pseudocodigo metodo:
class Puerta {
constructor(color,material){
this.color = color;
this.material = material;
}
nombre(parametros){
hacer algo
}
}

ahora con javaScript real:

javaScript metodo:

class Puerta {
constructor(color,material){
this.color = color;
this.material = material;
}
describir(){
console.log(`la puerta es de color ${this.color} y esta hecho de ${this.material}`)
}
}

para llamar a un metodo es igual a una funcion pero debemos asignarlo a un objeto.

pseudocodigo

pseudocodigo llamar:
objeto.metodo()

ahora con javaScript:

javaScript llamar:
puerta.describir();
resultado: "la puerta es de color negro y esta hecho de madera"

de esta manera llamamos a un metodo recuerda siempre debe de ir acompañado de un objeto previamente creado
