funciones

iniciamos con pseudocodigo

pseudocodigo funcion:
funcion nombre ( parametros ) {
hacer algo.
}
pseudocodigo funcion flecha:
nombre = ( parametros ) => {
hacer algo.
}

llamar a una funcion.

pseudocodigo llamar:
nombre de la funcion( parametros );

ahora con javaScript.

javaScript funcion:
function sumar(a,b){
console.log(a + b);
}
sumar(5,5)
javaScript funcion flecha:
const sumar=(a,b)=>{
console.log(a + b);
}
sumar(10,3)
