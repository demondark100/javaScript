operador ternario

ejemplo 1

let fruta = "naranja";
if (fruta == "naranja") console.log("es una deliciosa naranja");
else console.log("no es una naranja")
let fruta = "naranja";
(fruta == "naranja")
?
console.log("es una naranja")
: console.log("no es una rananja")

si queremos hacer mas de una cosa o afirmacion se debe poner entre parentesis.

ejemplo:

let fruta = "naranja";
(fruta == "naranja")
?
(
console.log("es una naranja"),
console.log("definitivamente es una naranja")
)
:
(
console.log("no es una rananja"),
console.log("definitivamente no es una naranja")
)

la ventaja de esto es que usa menos recursos que un if.

operador spread

sin operador spread

let valor1 = "valor 1";
let valor2 = "valor 2";
let valor3 = "valor 3";
console.log(valor1,valor2,valor3);
// es mucha cosa pero se puede simplificar mas con el operador spread.

con operador spread

let arr = ["valor 1","valor 2","valor 3"];
console.log(...arr)

con rest tambien podemos destructurar de forma mas sencilla a los arreglos.

ejemplo:

let arr1 = ["manzana","pera","naranja"];
let arr2 = ["sandia","melon","fresa"];
arr1.push(...arr2);
console.log(arr1)

y si queremos concatenar arreglos.

ejemplo:

let arr1 = ["manzana","pera","naranja"];
let arr2 = ["sandia","melon","fresa"];
let arr3 = [...arr1,...arr2]
console.log(arr3)
