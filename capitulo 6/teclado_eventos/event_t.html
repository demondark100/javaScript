eventos de teclado

HTML:

input type="text" class="input_tecla" placeholder="escribir">

javaScript:

const input_tecla = document.querySelector(".input_tecla");
input_tecla.addEventListener("keydown",()=>{
alert("xd");
})

con "keydown" el evento se activara cuando una tecla sea presionada.

resultado:

presiona una tecla en donde dice "escribir"

input type="text" class="input_tecla" placeholder="escribir">

const input_tecla = document.querySelector(".input_tecla");
