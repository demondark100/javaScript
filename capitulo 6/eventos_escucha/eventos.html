eventos de escucha

HTML:
button class="boton"> saludar /button>

pseudocodigo: 1:

elementoHTML.metodoEscucha("tipo de evento",()=>{
alert("hola")
})

pseudocodigo 2:

elementoHTML.metodoEscucha("tipo de evento",funcionEvento);
funcion funcionEvento(){
alert("hola")
elementoHTML.removerEvento("tipo de evento",funcionEvento)
}

javaScript_1:

const boton = document.querySelector(".boton");
boton.addEventListener("click",()=>{
alert("hola")
})

esto nos servira para hacer un evento de escucha sirve para cualquier tipo de elemento.

javaScript_2:

const boton = document.querySelector(".boton");
boton.addEventListener("click",saludo) function saludo(){
alert("hola");
boton.removeEventListener("click",saludo)
}

de esta manera podemos remover un elemento una vez se haya ejecutado.
