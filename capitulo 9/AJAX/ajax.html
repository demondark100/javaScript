AJAX

para esto abriremos un archivo txt y un servidor local usando: xampp esto para poder enviar y recibir peticiones y ademas porque es compatible con una gran cantidad de sistemas operativos.

una vez instalado abrimos las 2 primeras opciones y luego nuestros archivos donde trabajaremos lo ponemos en una carpeta llamada "htdocs" puedes buscarlo en tu buscador de tu sistema operativo.

javaScript:

sintaxis de objeto "XMLHttpRequest";

const peticion = new XMLHttpRequest;

con "XMLHttpRequest" podemos enviar peticiones "GET" , "POST" , etc.

archivo txt:

nombre del archivo txt "datos.txt":

{
"musica": "fiesta pagana",
"genero": "rock"
}

abrir archivo y metodo:

pseudocodigo:

const peticion = new XMLHttpRequest;

peticion.abrir("metodo","url archivo o sitio web");

javaScript;

const peticion = new XMLHttpRequest;

peticion.open("GET","datos.txt");

con "open" abriremos el archivo o url para poder enviar o recibir datos.

enviar datos:

pseudocodigo:

const peticion = new XMLHttpRequest;

peticion.abrir("metodo","url archivo o sitio web");
peticion.enviar()

javaScript:

const peticion = new XMLHttpRequest;

peticion.open("GET","datos.txt");
peticion.send()

con "send()" podemos enviar los datos.

metodos de ajax

const peticion = new XMLHttpRequest;
peticion.addEventListener("readystatechange",()=>{
console.log(peticion.readyState);
console.log(peticion.response)
console.log(peticion.status)
})
peticion.open("GET","datos.txt")
peticion.send()

con "readystatechange" abriremos el evento para poder trabajar con los datos.

con "readyState" veremos los estados.

estado 1: la solicitud se creo correctamente.
estado 2: la solicitud se envio correctamente.
estado 3: la solicitud se esta procesando.
estado 4: todo esta listo y ya se puede usar el resultado.
con "response" mostraremos el resultado en este caso:
{
"musica": "fiesta pagana",
"genero": "rock"
}
con "status" veremos el estado del sitio un ejemplo claro 404 y asi.

ejemplo

readystatechange

const peticion = new XMLHttpRequest
peticion.addEventListener("readystatechange",()=>{
if(peticion.readyState == 4 &&
peticion.status == 200
){
console.log(peticion.response)
}
})
peticion.open("GET","datos.txt")
peticion.send()

como nos damos cuenta con "readyState" tendremos que hacer muuchas validaciones pero se puede evitar con "load".

load

const peticion = new XMLHttpRequest
peticion.addEventListener("load",()=>{
console.log(peticion.response)
})
peticion.open("GET","datos.txt")
peticion.send()

nos mostrara el mismo resultado solo que nos evitamos las validaciones y es mas rapido.
