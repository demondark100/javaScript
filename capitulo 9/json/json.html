JSON

estos son como los objetos solo que se le agregan comillas.

const objeto = {
"nombre": "puerta",
"forma": "cuadrada",
"colo": "cafe"
}

se puede llamar al valor que queramos con los metodos que ya habiamos aprendido.

const objeto = {
"nombre": "puerta",
"forma": "cuadrada",
"colo": "cafe"
}
console.log(objeto.nombre);
console.log(objeto["nombre"]);

a la hora de trabajar con apis necesitamos descerializar y serializar los datos JSON.

serializar:

const objeto = {
"nombre": "puerta",
"forma": "cuadrada",
"colo": "cafe"
}
JSON.stringify(objeto)
console.log(JSON.stringify(objeto));

resultado:

{"nombre":"puerta","forma":"cuadrada","colo":"cafe"}

con "JSON.stringify()" podemos serializar osea convertir al objeto json en una string para que se puedan enviar los datos correctamente.

pero a la hora de recibir datos del servido o apis recibimos los datos serializados osea.

{"nombre":"puerta","forma":"cuadrada","colo":"cafe"}

por eso podemos descerializar un objeto json de esta manera para poder trabajar con esos datos.

const objeto = {
"nombre": "puerta",
"forma": "cuadrada",
"colo": "cafe"
}
let objetSerializado = JSON.stringify(objeto)
console.log(JSON.parse(objetSerializado));

con "JSON.parse()" podemos descerializar los objetos para tranajar con ellos.
