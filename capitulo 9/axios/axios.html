axios

axios es una libreria que nos facilita mas el uso de "ajax" y "fetch"

importante

para instalar esta libreria necesitamos pegarlo antes de tu archivo js de tu archivo HTML para evitar errores no deseados.

ejemplo

script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"> /script>
script src="archivo.js"> /script>

recordatorio axios trabaja con el metodo "GET" por defecto.

axios("https://reqres.in/api/users/2")
.then(res=>console.log(res))

con esto ya desencapsulamos toda la informacio json.

ahora si queremos trabajar con otro metodo hay dos formas de hacerlo.

forma1: (recomendado)

axios.post("https://reqres.in/api/users/2",{
"music": "A Sky Full of Stars",
"artista": "Coldplay"
})
.then(res=>console.log(res))
axios("https://reqres.in/api/users/2",{
method: "post",
data: {
"music": "A Sky Full of Stars",
"artista": "Coldplay"
}
})
.then(res=>console.log(res))
