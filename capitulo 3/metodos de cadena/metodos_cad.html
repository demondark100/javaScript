metodos de cadena

hay muchos metodos que nos facilitaran las cosas.

iniciamos:

let cadena = "hola mundo"

cadena.startsWith("h")
resultado: true
cadena.startsWith("a")
resultado: false

esto nos servira para saber si una cadena inicia con un caracter en especifico

let cadena = "mundo"

cadena.endsWith("o");
resultado: true
cadena.endsWith("e");
resultado: false

esto nos servira para saber si una cadena finaliza con algun caracter en especifico.

let cadena = "hola mundo XD";

cadena.includes("l")
resultado: true
cadena.includes("z")
resultado: false

esto nos servira para saber si una cadena contiene un caracter en especifico a diferencia de las dos primeras a este no le importara la posision.

let cadena = "hola mundo";

cadena.indexOf("a")
resultado: 3

esto nos mostrara la posision de un caracter en especifico

let cadena = "vivir es vida";

cadena.lastIndexOf("v")
resultado: 9

esto nos servira para saber la ultima posision de un caracter en especifico.

let cadena = "123";

pseudocodigo:
variable.metodo ( parametro 1 longitud de cadena , parametro 2 cadena que queremos poner );

javaScript:
cadena.padStart(4,0)
resultado: 0123

esto nos servira para rellenar una cadena al imicio

let cadena = "123";

pseudocodigo:
variable.metodo ( parametro 1 longitud , parametro 2 cadena que queremos poner al final );

javaScript:
cadena.padEnd(4,4)
resultado: "1234"

esto nos servira para rellenar una cadena al final.

let cadena = "hola";

cadena.repeat(3)
resultado: "hola","hola","hola"

esto nos servira para que el contenido de una cadena de texto se repita cuantas veces queramos

let cadena = "hola mundo";

cadena.split(" ");
resultado: ["hola","mundo"]

esto nos servira para separar una cadena y se convertira en un arreglo

let cadena = "hola mundo";

pseudocodigo:
crear_variable = variable.metodo ( parametro 1 desde donde inicia , parametro 2 hasta donde finaliza y no cuenta el ultimo numero );

javaScript:

let subCadena = cadena.substring(5,10);

resultado: "mundo"

esto nos servira para crear una subcadena a diferencia de los otros metodos esto si necesita una variable de respaldo para que pueda guardar la nueva cadena de texto.

cadena.toUpperCase();

cadena.toLoweCase();

esto nos servira para convertir una cadena de texto a mayuscula o minuscula

con "toUpperCase" convertiremos la cadena a mayuscula.
con "toLoweCase" convertiremos una cadena a minuscula.

let cadena = 123;

cadena.toString()
resultado: "123"

esto nos servira para convertir cualquier dato a una cadena de texto

let cadena = " hola mundo ";

cadena.trim();
resultado: "hola mundo"

esto nos servira para quitar los espacios al principio y al final de una cadena de texto.

con "trimStart()" quitamos los espacios del principio de una cadena.
con "trimEnd()" quitamos el espacio al final de una cadena.
