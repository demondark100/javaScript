metodos de arreglos

los metodos de arreglos nos facilitaran las cosas.

let frutas = ["manzana","pera","naranja"];

let frutas2 = frutas.pop();

resultado frutas: ["manzana","pera"]; resultado frutas2: "naranja";

esto nos servira para quitar el ultimo elemento de un arreglo.

let frutas = ["manzana","pera","naranja"];

let frutas2 = frutas.shift();

resultado frutas: ["pera","naranja"];
resultado frutas2: "manzana";

esto nos servira para quitar el primer elemento de un arreglo

let frutas = ["manzana","naranja"];

frutas.push("pera");
resultado: [
"manzana",
"naranja",
"pera"
];

esto nos servira para agregar elementos al final de un arreglo

let frutas = [
"naranja",
"pera"
];

frutas.unshift("manzana");
resultado: [
"manzana",
"naranja",
"pera"
]

esto nos servira para agregar elementos al principio de un arreglo

let abecedario = [
"a",
"b",
"c",
"d"
];

abecedario.reverse()
resultado: [
"d",
"c",
"b",
"a"
];

esto nos servira para poner en reversa un arreglo
