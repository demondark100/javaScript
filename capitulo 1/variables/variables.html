pseudocodigo

codigo javaScript

var number1 = 15;
let number2 = 15;
const number3 = 15;

con "var" , "let" , "const" declaramos variables.

no es recomendable usar var porque es una declaracion global.

"let" es la forma correcta de declarar variables.

"const" si declaramos variables con const su valor no cambia.
