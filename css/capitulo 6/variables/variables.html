variables

las variables nos sirve para guardas estilos en css.

pseudocodigo crear variable:
--un-rojo: #f00;

pseudocodigo llamar variable:
var(--un-rojo);

variables globales

las variables globales nos permitira usar la variable en cualquier elemento que queramos.
para usarlas es necesario crear un "root".
