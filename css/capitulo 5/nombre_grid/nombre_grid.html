con nombrar lineas

para poder nombrar lineas es necesario hacerlos desde el contenedor en cada medida que hagamos es necesario que los nombres esten dentro de [] y no deben llevar comillas.

pseudocodigo:

grid-template-columns:
[nombre de linea]
medida.
[nombre de linea]
medida
[nombre de linea]
medida
[nombre de linea]
;


css:

grid-template-columns:
[linea_1]
110px.
[linea_2]
110px
[linea_3]
110px
[linea_4]
;
