grid template

con esto podremos definir cuantas filas y columnas queremos , estos se aplican a los grid container.

grid-template-rows

con esto diremos cuantas filas queremos en nuestro grid content.
pseudocodigo:
grid-template-rows: cuantas filas quieresd y con que medida.
ejemplo:
grid-template-rown: 10px 10px 10px 10px 10px;
en este caso creamos 5 filas con una medida de 10px cada una.
si tienes menos cantidad de elementos hijos que la cantidad de gird que hayas declarado siempre se dibujaran los grids que declaraste , esto se aplica a las columnas tambien.
