clases y metodos

podemos hacer muchas cosas con las clases de los elementos. "classList"

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.add(":v");

con "add" podremos agregar una clase a un elemento html.

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.remove("xd");

con "remove" podremos quitarle una clase a un elemento.

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.item(0);

con "item" nos devolvera el indice de la clase que le indiquemos en este caso [0] = "xd".

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.contains("xd");
resultado: true:
xd.classList.contains("a");
resultado: false

con "contains" verificarenos si un objeto lleva una clase o no es como un "includes".

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.replace("xd","a");

esto nos servira para reemplazar una clase por otra.

en el primer parametro ponemos la clase que queramos remplazar y en la segunda pondremos la clase que queramos agregar.

HTML:
p class="xd">xd /p>
javaScript:
const xd = document.querySelector(".xd");
xd.classList.toggle("a");

con "toggle" verificaremos si el elemento tiene una clase y si no la tiene la agregara y si la tiene la eliminara.
