obtencion y modificacion de childs

con esto podremos ver lo que hay en un elemento html.

HTML:
p class="xd"> hola b> mundo /b> /p>
javaScript:
const xd = document.querySelector(".xd");
xd.textContent;
resultado: "holamundo"

con "textContent" nos mostrara solo el contenido de un elemento.

HTML:
p class="xd"> hola b> mundo /b> /p>
javaScript:
const xd = document.querySelector(".xd");
xd.innerHTML;
resultado: "hola b> mundo /b>"

con "innerHTML" nos mostrara las etiquetas qye esten dentro de ese elemento.

HTML:
p class="xd"> hola b> mundo /b> /p>
javaScript:
const xd = document.querySelector(".xd");
xd.outerHTML;
resultado: "p class="xd"> hola b> mundo /b> /p>"

con "outerHTML" nos mostrara todo el elemento html.
