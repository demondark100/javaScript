modicacion childs

HTML:
div class="contenedor"> /div>
javaScript:
let parrafo = document.createElement("P");
let fragmento = document.createDocumentFragment();
parrafo.textContent = "hola mundo";
fragmento.appendChild(parrafo);
contenedor.appendChild(fragmento);

con "appendChild()" hacemos que un elemento creado en javaScript sea puesto como hijo a un elemento HTML ya creado.

HTML:
div class="a_content">
p> algo 1 /p>
p> algo 2 /p>
p> algo 3 /p>
/div>
javaScript:
const a_content = document.querySelector(".a_content");
let algo1 = a_content.firstElementChild;
let algo2 = a_content.lastElementChild;

con "firstElementChild" y "lastElementChild" nos mostrara el primer y ultimo hijo del contenedor

HTML:
div class="a_content">
p> algo 1 /p>
p> algo 2 /p>
p> algo 3 /p>
/div>
javaScript:
const a_content = document.querySelector(".a_content");
let algo1 = a_content.children[];

con "children[]" podremos ver los hijos en la posision que queramos.
