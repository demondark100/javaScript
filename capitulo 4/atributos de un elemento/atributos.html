cambiar atributos

HTML:
input class="a" type="text"> javaScript:
let a = document.querySelector(".a");
a.setAttribute("type","color");

con esto podremos cambiar el atributo de un elemento

HTML:
input class="a" type="text"> javaScript:
let a = document.querySelector(".a");
a.getAttribute("type");

esto nos servira para acceder al atributo dde un nodo

HTML:
input class="a" type="text"> javaScript:
let a = document.querySelector(".a");
a.getAttribute("type");
HTML:
input class="a" type="text"> javaScript:
let a = document.querySelector(".a");
a.removeAttribute("type");

con esto removeremos un atributo de un elemento
