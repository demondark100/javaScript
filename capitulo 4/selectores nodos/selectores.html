selectores de elementos

estos nos permitiran seleccionar el elemento o el grupo de elementos que queramos seleciionar.

HTML:
p id="elemento">/p>
javaScript:
let parrafo = document.getElementById("elemento")

esto nos permitira seleccionar los elementos atravez de su id

HTML:
p>/p>
p>/p>
javaScript:
let parrafo = document.getElementsByTagName("p")

esto nos servira para seleccionar un grupo de elementos en este caso todos los "p".

HTML:
p class="elemento">/p>
javaScript:
let parrafo = document.querySelector(".elemento")

esto nos servira para seleccionar cualquier tipo de elemento id o clases.

HTML:
p class="elemento">/p>
p class="elemento">/p>
javaScript:
let parrafo = document.querySelectorAll(".elemento")

esto nos sevira para seleccionar a un grupo de elementos que tengan la misma clase.
