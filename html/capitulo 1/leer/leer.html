importante

para poder abrir la consola de desarrollador en el navegador se debe presionar F12

para poder "inspeccionar" un elemento en especifico es necesario hacer click derecho sobre el elemento que queramos y darle en inspeccionar , automaticamente se abrira la consola de desarrollador con el elemento html que seleccionamos.

un ejemplo has clock derecho sobre el boton que dice hola mundo y dale en inspeccionar.

como te das cuenta podras ver al elemento "button" y de paso como esta estructurado este sitio web esto se puede hacer en todos los sitios web.

para crear un archivo html es necesario poner la extencion ".html" ejemplo

para cuando sea el sitio web principal como por ejemplo el menu principal o el lobby por asi decirlo es necesario ponerle "index" ejemplo.

por ultimo descarga un editor de codigo recomendable " visual studio code" o "sublime text" cualquiera de los dos editores de codigo son perfectos.
